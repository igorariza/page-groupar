## Comenzando 🚀

_Crea una carpeta nueva para el proyecto y Clona el repositorio_

```
git clone https://github.com/igorariza/page-groupar.git
```

Mira **Deployment** para conocer como desplegar el proyecto.


### Pre-requisitos 📋

_Npm_
_Nodejs_

```
npm --version
node --version

npm install
npm start
```
